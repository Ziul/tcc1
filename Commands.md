
Instruções Básicas
======== 

APT
----

- Install

	apt-get install git


- Remove

	apt-get purge git


- Search

	apt-cache search git

YUM
----

- Install

	yum install git


- Remove

	yum remove git


- Search

	yum search git

Portage (emerge)
----

- Install

	emerge git


- Remove

	emerge -C git


- Search

	emerge -s git


PKGng
----

- Install

	pkg install git


- Remove

	pkg delete git


- Search

	pkg search git
